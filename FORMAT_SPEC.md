# 短剧剧本与大纲格式规范 v1.0

> 对标一线广播电视剧本行业标准。适用于 1-3 分钟竖屏微短剧。

---

## 一、总则

### 1.1 格式铁律
- **每个场景独立成段**，段间空一行
- **不滥用 `|` 符号**：`|` 只用于表格分隔
- **层级清晰**：用 Markdown 标题 (`#`, `##`, `###`) 表达层级
- **每行不超过 80 字符**（专业编剧软件行业惯例）
- **不使用 `**加粗**` 标记对白**（对白用 `角色名（情绪）："对白"`，加粗仅用于强调）

### 1.2 排版工具
- 必须使用 **Markdown 语法**（不要用 Word/HTML）
- 标题：`#` 一级（集） / `##` 二级（场景） / `###` 三级（节拍）
- 列表：用 `-` 或 `1.`（数字编号）
- 引用：用 `>` （旁白/内心独白）
- 分隔：用 `---`（场景间分隔）
- 表格：用 `|` `---` `|`（**仅在制作拍摄表时使用**）

## 二、剧本格式规范

### 2.1 集标题

**正确**：
```markdown
# 第一集：才女入乐籍

# 第一集《才女入乐籍》

# Episode 1: 才女入乐籍
```

**错误**：
```markdown
# 第一集：（只写了标点）
# 才女入乐籍（缺"第N集："前缀）
```

### 2.2 场景标题（场号制）

**格式**：`## 场景 N | 地点 | 时间 | 内/外景`

**正确**：
```markdown
## 场景 1 | 浣花溪畔 | 黄昏 | 外景

## 场景 2 | 乐坊门前 | 黄昏 | 外景

## 场景 3 | 宴席上 | 夜晚 | 内景
```

**错误**：
```markdown
## 场景一（必须用阿拉伯数字）
## 场景1 浣花溪畔（缺分隔符 `|`）
## 浣花溪畔（缺"场景 N |"）
```

### 2.3 场景描述

**格式**：`> 【景别/运镜】画面内容`

**正确**：
```markdown
> 【竖屏构图，特写】俯拍落花飘入溪水，特写花瓣上的水珠。

> 【中景，平视】薛涛（20 多岁）蹲在溪边浣花，素衣清雅。

> 【动态跟拍】薛涛被带入乐籍，更换官妓服饰，眼神倔强。
```

**错误**：
```markdown
> 浣花溪畔（缺【景别/运镜】前缀）
【特写】俯拍落花（缺 `>` 引用标记）
```

### 2.4 人物对白

**格式**：`**角色名**（情绪/动作）："对白内容"`

**正确**：
```markdown
**薛涛**（抬头，目光坚定）："诗在我在，你们夺不走的。"

**官差甲**（轻蔑）："乐妓薛涛，从今往后，你的诗只配给老爷们助酒兴。"

**群众**（画外音，嘲讽）："一个女妓，还敢自称诗人？简直笑话！"
```

**错误**：
```markdown
薛涛：诗在我在（缺 **加粗** + 情绪/动作 描述）
**薛涛**：诗在我在（情绪描述用括号，不是中文括号）
薛涛（坚定）：诗在我在（缺"加粗"，缺引号包裹对白）
```

### 2.5 旁白 / 内心独白

**格式**：`> 旁白（身份，状态）："内容"`

**正确**：
```markdown
> 旁白（女声，低沉）："大历年间，成都浣花溪。一个被贬入乐籍的女诗人，在为落花低泣。"

> 内心独白（薛涛）："长安虽大，何处是我容身之所？"
```

### 2.6 动作指示

**格式**：单独成行，描述动作，**不加引号**

**正确**：
```markdown
薛涛提笔在纸上写诗，字迹秀美飘逸。

韦皋拍案而起，屏息静听。
```

**错误**：
```markdown
"薛涛提笔在纸上写诗"（动作加引号是错的）
**薛涛**："提笔写字"（动作不是对白）
```

### 2.7 音效 / 音示

**格式**：`* 音效：磨墨声、赞叹声`

**正确**：
```markdown
* 音效：磨墨声、赞叹声
* 音乐：古琴低鸣
* 静默 3 秒
```

### 2.8 完整剧本示例

```markdown
# 第一集：才女入乐籍

## 场景 1 | 浣花溪畔 | 黄昏 | 外景

> 【竖屏构图，特写】俯拍落花飘入溪水，特写花瓣上的水珠。

> 旁白（女声，低沉）："大历年间，成都浣花溪。一个被贬入乐籍的女诗人，在为落花低泣。"

## 场景 2 | 乐坊门前 | 黄昏 | 外景

> 【中景，平视】薛涛（20 多岁）被两个官差押至乐坊门前，衣衫素雅，眼神倔强。

**官差甲**（轻蔑）："乐妓薛涛，从今往后，你的诗只配给老爷们助酒兴。"

**群众**（画外音，嘲讽）："一个女妓，还敢自称诗人？简直笑话！"

**薛涛**（抬头，目光坚定）："诗在我在，你们夺不走的。"

---

# 第二集：百折不挠

## 场景 1 | 薛涛书房 | 深夜 | 内景

> 【特写】薛涛双手布满伤痕，桌上堆满废纸。
```

## 三、大纲格式规范

### 3.1 大纲结构

**正确**：
```markdown
# 《红笺女皇》大纲

## 故事立意
一句话正能量主题：歌颂唐代女诗人薛涛在困境中不屈不挠、以才华书写人生的精神。

## 主题关键词
- 时代：唐代中后期
- 人物：薛涛（女诗人，20-40 岁）
- 核心：才华、坚韧、家国情怀
- 情绪：温暖、励志

## 主要人物
- **薛涛**：女主人公，唐代女诗人，从乐妓到女校书
- **韦皋**：剑南节度使，薛涛的恩主与情感纠葛对象
- **元稹**：唐代大诗人，薛涛的知己恋人

## 故事线

### 第一幕：才女入乐籍（0-30 秒）
薛涛被贬入乐籍，从才女沦为乐妓，但她始终没有放弃对诗歌的热爱。

### 第二幕：百折不挠（30-60 秒）
薛涛在乐坊中坚持创作，屡败屡试，终于制成"薛涛笺"。

### 第三幕：纸短情长（60-90 秒）
薛涛与元稹相恋，以诗传情，但因元稹贬谪而分离。

### 第四幕：女校书（90-120 秒）
韦皋奏请朝廷，封薛涛为女校书，她用才华改写人生。

## 关键场景
- 场景 1：浣花溪畔（薛涛浣花）
- 场景 2：乐坊门前（薛涛入籍）
- 场景 3：薛涛书房（薛涛制笺）
- 场景 4：长安街头（薛涛与元稹相遇）
```

### 3.2 大纲层级规则

| 层级 | 语法 | 用途 | 示例 |
|------|------|------|------|
| L1 | `#` | 大纲名称 | `# 《红笺女皇》大纲` |
| L2 | `##` | 主要板块 | `## 故事立意` / `## 主要人物` / `## 故事线` |
| L3 | `###` | 幕/节拍 | `### 第一幕：才女入乐籍` |
| L4 | `-` | 列表项 | `- **薛涛**：...` |

### 3.3 `|` 符号使用规范

**适用场景**：
- 拍摄计划表（集-场-景-时间-内容）
- 场次信息密集表达（如 `## 场景 1 | 浣花溪畔 | 黄昏 | 外景`）

**禁止使用**：
- ❌ 用于分隔**对白片段**（如 `对白："诗在我在" | 音效：磨墨声`）
- ❌ 用于分隔**不相关的字段**（如 `时长 | 内容` 后接 5 个段落）
- ❌ 用于分隔**节拍说明**（如 `1. 时间推移 | 0-15s | 双手布满伤痕`）

**错误 vs 正确**：

```markdown
❌ 错误（滥用 |）：
| 时间 | 运镜 | 画面内容 | 音效/对白 |
|---|---|---|---|
| 0-5s | 特写 | 落花飘入溪水 | 旁白：大历年间... |

✅ 正确（用 Markdown 表格或分段落）：
### 节拍 1：开场落花（0-5s）
- **景别**：竖屏构图，特写
- **画面**：俯拍落花飘入溪水
- **音效**：流水声、鸟鸣
- **旁白**："大历年间，成都浣花溪。"
```

## 四、符号使用对照表

| 符号 | 正确用法 | 错误用法 |
|------|---------|---------|
| `\|` | 场次信息分隔 / 表格 | 段落内分隔 |
| `>` | 旁白 / 场景描述 | 强调普通对白 |
| `**` | 角色名 | 强调对白内容 |
| `*` | 音效 / 音乐 | 强调对白内容 |
| `-` | 无序列表 | 长段落断行 |
| `1.` | 有序步骤 | 长段落断行 |
| `#` | 标题 | 装饰性强调 |
| `---` | 场景间分隔 | 段落装饰 |

## 五、检查清单

### 剧本检查清单
- [ ] 集标题用 `# 第N集：标题`
- [ ] 场景标题用 `## 场景 N | 地点 | 时间 | 内/外景`
- [ ] 场景描述用 `> 【景别】内容`
- [ ] 对白用 `**角色**（情绪）："对白"`
- [ ] 旁白用 `> 旁白（身份）："内容"`
- [ ] 场景间用 `---` 分隔
- [ ] 没有滥用 `|`
- [ ] 没有把动作加引号

### 大纲检查清单
- [ ] 顶层用 `# 《片名》大纲`
- [ ] 主要板块用 `##`
- [ ] 幕/节拍用 `###`
- [ ] 人物/场景列表用 `-`
- [ ] `|` 仅用于场次信息或拍摄表
- [ ] 层级不超过 3 级
